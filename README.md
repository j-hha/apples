# Web Component Charts
## Summary
Charts as reusable web components - a practice project.
## Usage
### In HTML
```
<component-name data-json='json-file-name'>
        <span slot="caption-text">caption text</span>
</component-name>
```
### Available Components
- `<bar-chart>`
